# ClockingIn
- 项目名称：手机考勤客户端
- 创建时间：2019.2.27
- 上次更新时间：2019.5.9
-文档状态：编辑中

>网工三班第八组

### 主要内容
需要先连接server端，从server端拉去教师上课表。
老师使用该app可以实现学生上课打卡，身份认证，出勤统计并且数据可视化，并且可以根据调课来为每一节课增添课程。
最终目的达到解决学生上课时候的考勤问题。
